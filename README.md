# WebdriverIO Test Framework

A sample test case for CLICKDOCpro

## Run Locally

Clone the project

```bash
  git clone https://github.com/mhikmetozcan/CGM.git
```

Go to the project directory

```bash
  cd cgm
```

Start the server

```bash
  npm run wdio
```
